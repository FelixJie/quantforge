
# 产业链投研看板 - 实现计划

## 1. 概述

创建一个通用的产业链投研看板系统，支持多主题/概念选择（如人形机器人、固态电池、AI等），包含AI智能分析功能，后台运行分析任务，完成后展示结果。

## 2. 核心功能

### 2.1 主题/概念选择
- 支持选择不同产业链主题
- 预置热门概念（人形机器人、固态电池、AI等）
- 支持自定义概念/行业搜索

### 2.2 多Tab页结构（如示例图）
- 总览：产业全景概览
- 成本构成：BOM成本图表
- 各零部件子Tab：减速器、丝杠、电机、传感器等
- 替代风险：供应链分析
- 估值全景：行业估值分析

### 2.3 AI智能分析
- 自动搜集相关研报
- AI分析产业链逻辑
- 生成投资观点摘要
- 后台任务执行，完成后展示

## 3. 实现方案

### 3.1 后端增强

1. **research.py 新增功能**
   - 获取概念板块成分股
   - 获取产业链相关研报
   - AI分析任务接口（异步）
   - 任务状态查询和结果获取

2. **新增后端任务模块**
   - 后台任务调度器
   - AI分析任务队列
   - 结果缓存机制

### 3.2 前端看板视图

创建 `IndustryResearchDashboard.vue` 视图，包含：

1. **顶部区域**
   - 主题选择器
   - 数据更新时间
   - 市场热度指标
   - AI分析任务入口

2. **Tab导航**
   - 总览
   - 成本构成
   - 减速器
   - 丝杠
   - 电机
   - 传感器
   - 腿海/材料
   - 替代风险
   - 估值全景

3. **各Tab内容模板**
   - 产业全景卡片
   - 相关概念股表格
   - BOM成本面积图
   - 机构研报列表
   - AI分析摘要
